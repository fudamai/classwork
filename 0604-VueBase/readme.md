# 博客网站

项目地址：[blog](./blog)

需求实现：

- 列表页、详情页分离。
- 侧边栏有个人信息、最火文章、最新评论
- 使用mock生成随机数据
- 各组件使用 vuex 进行数据交互
- 在列表页点击标题进入对应的详情页
